Two kinds of apps.

"Hosted App"

Existing Web Site + App Manifest = Hosted App

Pros

  1. Easy. takes very little work to convert!
  2. Works well with existing back-end frameworks (Django, Rails, etc.)
  3. Great for developers who aren't comfortable with front-end tech.

Cons

  1. Not that easy: where's the back button?
  2. Performance
    • Synchronous page-loads on every user action = slow.
    • Bandwidth hog!
      • Requires WiFi to work well
      • Bandwidth === $$$
  3. Little-to-no offline support - browser cache might save you...
  4. Poor first-run experience
  5. "Glorified Bookmark" :(

"Packaged App"

.zip of Static resources (including App Manifest)

Pros

  1. No Hosting Required!
  2. Awesome first-run experience!
  3. Performance
    • No network requests for resources
  4. No Connection? No Problem.
  5. Extra APIs
    • Socket connections, SMS, Camera, Contacts
  6. "Feels Like Native!" <3
